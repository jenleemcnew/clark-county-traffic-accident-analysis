# Clark County, NV — Traffic Accident Analysis

## Overview
An exploratory data analysis of traffic accident data from Clark County, Nevada (2016).
This project investigates relationships between driver age, weather conditions, road 
conditions, time of day, and accident frequency and type using Python-based data 
analysis and visualization.

> **Note:** The source dataset was not fully normalized, which may introduce some 
> variability in the findings. Results should be interpreted as directional insights 
> rather than statistically conclusive.

---

## Key Questions
- Does driver age correlate with accident involvement or severity?
- How do weather conditions influence accident frequency?
- What role do road conditions play in accident outcomes?
- Are certain times of day associated with higher accident rates?

---
